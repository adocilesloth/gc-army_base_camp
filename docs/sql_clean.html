sql_abc_clean

Escapes the majority of punctuation n data going to the database by prefixing it with "\\". Also escapes common sql commands: select, drop, insert, update, join.

Arguments:
  • string - data to be inserted into the database (str)
Returns:
  • string - cleaned string (str)
