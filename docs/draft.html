abc_draft

Class that controls joinging, leaving and running the player draft.

Functions

  • draft_page
  • join_draft
  • leave_draft
  • draft_page
  • run_draft

draft_page

Generates the html content for joining or leaving the draft. If in the draft, the user sees the leave draft page, else sees the join draft page. Also allows using the army password to bypass the draft.

Called by:
  • controller/main->handle

join_draft

Adds the user to abc_users table. During this, records the current phpBB rank_id into the user_soldierid column in the abc_users table.

Calls:
  • include/abc_sql_clean->sql_abc_clean
  • include/abc_sql_clean->sql_abc_unclean
Called by:
  • controller/main->handle

leave_draft

Deletes the user to abc_users table.

Called by:
  • controller/main->handle

draft_list

Generates the html content for showing the draft list.

Calls:
  • core/permissions->whitelist
  • include/abc_sql_clean->sql_abc_unclean
Called by:
  • controller/main->handle

run_draft

Runs the draft and assignes users to the selected armies. Multiple users can be assigned to multiple armies (one per user) at a time.

Calls:
  • core/abc_draft->draft_list
Called by:
  • controller/main->handle
