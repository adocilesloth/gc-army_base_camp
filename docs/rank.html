abc_rank

Class that controls creating, editing, deleting ranks. Awarding ranks is handled by abc_army.

Functions

  • rank_list
  • add_rank
  • edit_rank
  • delete_rank
  • get_rank_path
  • get_rank_db_row

rank_list

Generates the html content for creating, editing and deleting ranks.

Calls:
  • core/abc_rank->get_rank_path
  • include/abc_sql_clean->sql_abc_unclean
Called by:
  • core/abc_battleday->add_rank
  • core/abc_battleday->edit_rank
  • core/abc_battleday->delete_rank
  • controller/main->handle

add_rank

Adds a rank to abc_ranks.

Calls:
  • core/abc_rank->rank_list
  • core/abc_rank->get_rank_path
  • include/abc_sql_clean->sql_abc_clean
Called by:
  • controller/main->handle

edit_rank

Edits a rank in abc_ranks.

Calls:
  • core/abc_rank->rank_list
  • core/abc_rank->get_rank_db_row
  • include/abc_sql_clean->sql_abc_clean
Called by:
  • controller/main->handle

delete_rank

Deletes a rank from abc_ranks. Does not delete the corresponding phpBB rank as this tends to break the other phpBB ranks.

Calls:
  • core/abc_rank->rank_list
  • core/abc_rank->get_rank_db_row
Called by:
  • controller/main->handle

get_rank_path

Generates the path for saving rank icons.

Arguments:
  • rank_path - path to rank icon folder, set in this function (str, reference)
  • army_id - The offset from SBT, set in this function (int, reference)
Called by:
  • core/abc_rank->add_rank
  • core/abc_rank->get_rank_db_row

get_rank_db_row

Gets the data base row from abc_ranks for the battle selected by rank_radio.

Arguments:
  • rank_path - path to rank icon folder (str, reference)
  • army_id - The offset from SBT (int, reference)
Returns:
  • row - database row (dict)
  • False - On failure (bool)
Calls:
  • core/abc_rank->get_rank_path
Called by:
  • core/abc_rank->edit_rank
  • core/abc_rank->delete_rank
