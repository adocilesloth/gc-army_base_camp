Class that controls the database side of starting the campaign. Forums are dealt with in abc_forum.

start_page

Generates the html content for starting the campaign.

Calls:
  • include/abc_sql_clean->sql_abc_unclean
Called by:
  • controller/main->handle

start_campaign

Starts the campaign:
Sets the phpBB extension configuration for the campaign.
Creates the phpBB forum user groups for Squaddies, Officers, High Command ,General (the General group is hidden) and Tournament Administrators.
Creates a new campaign in abc_campaigns.
Creates folders for this campaign's medal, rank and division images.
Creates two armies plus the Tournament Administrator army in abc_armies.
Creates General and New Recruit ranks for each army and Tournament Administrator rank in abc_ranks.
Creates New Recruit division all three armies in abc_divisions.
Assigns leaders of all three armies by adding them to the relavent phpBB groups and abc_users.
Assigns the relavent ranks to the Generals and Tournament Administrators in phpBB and abc_users.

Returns:
  • True - If success (bool)
  • False - If failure (bool)
Calls:
  • include/abc_sql_clean->sql_abc_clean
Called by:
  • controller/main->handle
