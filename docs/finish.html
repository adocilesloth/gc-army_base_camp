abc_finish

Class that controls the database side of the end of the campaign. Forums are dealt with in abc_forum. Does not delete the phpBB rank as this tends to break the other phpBB ranks.

Functions

  • finish_page

finish_page

Generates the html content for ending the campaign.

Called by:
  • controller/main->handle

finish_campaign

Finishes the campaign. Sets users' phpBB ranks back to the ranks they had before they entered the draft. Deletes the armies' phpBB groups. Removed the TA phpBB group from ABC admins. Resets the phpBB extension campaign settings back to defaults.

Called by:
  • controller/main->handle
