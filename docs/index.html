Introduction

Documentation for Army Base Camp (ABC), the software used by Global Conflict (GC) to administer campaigns. This documentation should be used with reference to the code. It is not designed to explain exactly how functions do thing, just what they do.

ABC uses php to interface with the phpBB forum software. This version of ABC was built using php v7.2 and phpBB v3.2.5 and these are the only versions that are known to function correctly. Other version of php may work (I imagine that newer php versions should be fine) and all phpBB v3.2.x versions should also work, but no guarentee is given.

This was created with the ethos that noone should have to ask anyone to do anything for them. So Generals can create forums, ranks, divisions, medals without needing to ask Tournament Administrators or Execs. Officers and up can promote anyone of a lesser rank upto their rank. Tournament Administrators can close and archive the campaign without needing an Exec. The only exception to this is that, once made, Generals cannot delete or edit forums. This is to prevent the loss of GC's history through accidental deletion of a forum.

The full code for ABC is avaliable as open source freeware and can be found on GitHub.

Lists of functions "Calls" and "Called by" exclude php and phpBB functions.

Legacy

Campaigns created before this version of ABC have differences in how data is stored in the same databases. Campaigns are considered legacy if campaign_id < 15. Armies are considered legacy if army_id < 40.
