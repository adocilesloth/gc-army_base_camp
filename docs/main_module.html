Sets default phpBB config settings and sends data to the extension settings page in the Admin Control Panel. Also has cleaning and uncleaning functions.

module

Sets default phpBB config settings and sends data to the extension settings page, abc_apc_body.html, in the Admin Control Panel.

Arguments:
  • id - Unused
  • mode - Unused
Calls:
  • acp/main_module->sql_abc_clean
  • acp/main_module->sql_abc_unclean

sql_abc_clean

Escapes the majority of punctuation n data going to the database by prefixing it with "\\". Also escapes common sql commands: select, drop, insert, update, join.

Arguments:
  • string - data to be inserted into the database (str)
Returns:
  • string - cleaned string (str)
Called by:
  • acp/main_module->module

sql_abc_unclean

Uncleans data comming from the database by removing all "\\"

Arguments:
  • string - data to be displayed from the database (str)
Returns:
  • string - uncleaned string (str)
Called by:
  • acp/main_module->module
