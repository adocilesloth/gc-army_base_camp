abc_menu

Class that creates the navigation navigation buttons (note, these are not links), user information and battleday signup histograms.

Functions

  • generate_menu
  • utc_to_sbt

generate_menu

Generates the html content for the navigation buttons, soldier information and battleday signup histograms. Histograms are not shown if no upcoming battle is scheduled, i.e. no battleday in the abc_battles has a battle_start value in the future.

Calls:
  • core/abc_menu->utc_to_sbt
  • core/permissions->whitelist
  • core/permissions->blacklist
Called by:
  • controller/main->handle

utc_to_sbt

Converts a UTC timestamp into the offset from SBT. For example, the timestamp for 19:00 UTC will return 1 in the winter. The function accounts for how GC does DST/Summer time.

Arguments:
  • utc - A UTC timestamp (int)
Returns:
  • sbt - The offset from SBT (int)
Called by:
  • core/abc_menu->generate_menu
