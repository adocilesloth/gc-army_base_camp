Functions

whitelist

Allows access if user in in one of the passed phpBB groups.

Arguments:
  • whitelist - array of phpBB group names the user needs to be a member of (str, optional)
Returns:
  • True - user is in a phpBB group in whitelist (bool)
  • False - user is not in a phpBB group in whitelist (bool)
Called by:
  • core/abc_army->army_list
  • core/abc_army->get_user_ids
  • core/abc_draft->draft_list
  • core/abc_menu->generate_menu

blacklist

Denies access if user in in one of the passed phpBB groups.

Arguments:
  • blacklist - array of phpBB group names the user needs to be not a member of (str, optional)
Returns:
  • True - user is not in a phpBB group in blacklist (bool)
  • False - user is in a phpBB group in blacklist (bool)
Called by:
  • core/abc_menu->generate_menu
