Database Tables

A number of tables are used within ABC. These control things like who is in what army, what the armies are called and so on. This version of ABC uses the tables from ABC2 to maintain the history. However, their usage is slightly different and is unintuitive at times.

abc_armies

This table holds the information for the armies.

Column name
Data Type
Comments
army_id
int(2)
Unique id of the army. Incremented by the code, not the database. Army is considered legacy if army_id < 40
campaign_id
int(2)
id of the campaign the army belongs to. Campaign is considered legacy if campaign_id < 15
army_name
varchar(33)
Name of the army
army_general
int(4)
phpBB user_id of the general (abc_user_id for legacy armies)
army_colour
varchar(6)
Army colour in HEX without the #
army_tag_structure
varchar(4)
Unused
army_tag
varchar(3)
Tag for the army. Tag structure is <army><division><rank>
army_join_pw
varchar(32)
Pasword to directly join an army and bypass the player draft
army_ts_pw
varchar(5)
Unused
army_is_neutral
int(1)
If army is neutral/Torunament Administrator. 1=True, 0=False
army_hc_forum_group
int(3)
phpBB group_id for the High Command forum group
army_officer_forum_group
int(3)
phpBB group_id for the Officer forum group
army_soldiers_forum_group
int(3)
phpBB group_id for the Squaddie forum group
army_time_stamp
int(10)
Timestamp of when the army was created

abc_battle_sign_ups

This table holds the information for users that have signed up to the battledays.

Column name
Data Type
Comments
sign_up_id
int(4)
Unique id of the sign up. Incrimented by the code, not the database.
battle_id
int(3)
battle_id that the user has signed up for
user_id
int(5)
phpBB user_id of the user that has signed up. For legacy campaigns, this is abc_user_id.
sign_up_hours
int(3)
Hours the user has signed up for. Is stored as 2h1 + 2h2 + ... + 2hn where hn is the hour they have signed up to. For example, if they have signed up for hours 1 and 2, this value would be 21 + 22 = 6
battle_time_stamp
int(10)
Timestamp of when the battle sign up was created
