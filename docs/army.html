abc_army

Class that controls the displaying of an army as well as the awarding of medals, ranks and divisions.

Functions

  • army_list
  • award_medal
  • award_rank
  • award_division
  • get_user_ids
  • generate_ribbon_signature

army_list

Creates the html content for displaying an army. Officers (and above) see a checkbox next to users of a lower rank. This allows them to award medals and assign ranks and divisions.

Calls:
  • core/permissions->whitelist
Called by:
  • controller/main->handle

award_medal

Awards medal(s) to selected user(s) by adding an entry into abc_medal_awards. Displays success or fail on return.

Calls:
  • core/abc_army->generate_ribbon_signature
  • include/abc_sql_clean->sql_abc_clean
Called by:
  • controller/main->handle

award_rank

Awards rank to selected user(s) by updating entry(s) in abc_users and phpBB's phpbb_users. Also add the user to the Officer phpBB group if necessary.

Calls:
  • core/abc_army->get_user_ids
Called by:
  • controller/main->handle

award_division

Assigns division to selected user(s) by updating entry(s) in abc_users. Also add the user to the High Command phpBB group if necessary.

Calls:
  • core/abc_army->get_user_ids
Called by:
  • controller/main->handle

get_user_ids

Gets user_id for all users of a specified user. Also gets the campaign_id and army_name. All three are collected by reference in the calling function.

Arguments:
  • user_ids - Array of user_id (int, reference)
  • campaign_id - Set by this function (int, reference)
  • army_name - Set by this function (str, reference)
Returns:
  • bool - True if success, else Fail.
Calls:
  • core/permissions->whitelist
Called by:

generate_ribbon_signature

Creates phpBB signatures out of the ribbons for awarded medals. Uses old ABC code.

Arguments:
  • user_ids - Array of user_id (int)
  • campaign_id - Campaign to generate signature for
Called by:
  • core/abc_army->award_medal
