abc_division

Class that controls creating, editing, deleting divisions. Awarding divisions is handled by abc_army.

Functions

  • division_list
  • add_division
  • edit_division
  • delete_division
  • get_division_path
  • get_division_db_row

division_list

Generates the html content for creating, editing and deleting divisions.

Calls:
  • core/abc_division->get_division_path
  • include/abc_sql_clean->sql_abc_unclean
Called by:
  • core/abc_battleday->add_division
  • core/abc_battleday->edit_division
  • core/abc_battleday->delete_division
  • controller/main->handle

get_division_path

Generates the path for saving division icons.

Arguments:
  • division_path - path to division icon folder, set in this function (str, reference)
  • army_id - The offset from SBT, set in this function (int, reference)
Called by:
  • core/abc_division->add_division
  • core/abc_division->get_division_db_row

get_division_db_row

Gets the data base row from abc_divisions for the division selected by division_radio.

Arguments:
  • division_path - path to division icon folder (str, reference)
  • army_id - The offset from SBT (int, reference)
Returns:
  • row - database row (dict)
  • False - On failure (bool)
Calls:
  • core/abc_division->get_division_path
Called by:
  • core/abc_division->edit_division
  • core/abc_division->delete_division
