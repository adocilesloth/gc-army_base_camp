Functions

show_history

Generates the html content for an army's history. If no argument is passed, if a campaign is running and the user has been assigned an army, the user's army is shown. Otherwise, the army with the lowest army_id in abc_armies in the latest campaign is shown.

Arguments:
  • army_id - army_id of army to show (int, optional)
Calls:
  • include/abc_sql_clean->sql_abc_unclean
